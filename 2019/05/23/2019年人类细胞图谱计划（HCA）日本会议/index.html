2019年人类细胞图谱计划（HCA）日本会议

发表于 2019-05-23 | 分类于 生信分析


  2019年HCA会议于五月23日，24日在日本举行，届时领域内知名科学家共聚一堂，探讨单细胞技术的发展，遇到的挑战及解决方案，是单细胞领域的一场盛宴。油管网站提供全程直播及后续录播服务，点击下方Video可直达视频地址（需科/学/上/网）。
